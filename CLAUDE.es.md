Las convenciones del proyecto para todos los agentes están en [AGENTS.es.md](AGENTS.es.md) — **léelas y síguelas desde el primer paso**.

En particular: sigue la sección **AI / agent workflow — standard git commits** —
inspecciona `git status` primero, trabaja directamente en el checkout actual (crea una rama
separada solo si el usuario lo pide explícitamente), preserva los cambios locales no relacionados, y haz commit
solo cuando el usuario lo solicite.
